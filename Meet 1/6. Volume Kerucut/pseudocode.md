## Pseudocode

```
PROSEDUR VOLUME_KERUCUT(diameter, tinggi)
    jari_jari <- diameter / 2
    PI <- 3.14
    volume <- (1/3) * PI * jari_jari * jari_jari * tinggi
    KEMBALIKAN volume
AKHIR PROSEDUR

// Program Utama
diameter <- 5
tinggi <- 3
hasil <- VOLUME_KERUCUT(diameter, tinggi)
TAMPILKAN hasil
```
